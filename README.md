# ClassificacaoDeTriangulo

![](https://i.imgur.com/EFJMxQR.gif)

## Projeto

O **Classificador de Triângulos** é uma aplicação em C# que a partir de 3 valores determina a classificação do triangulo formado por esses valores.

## Funcionalidades

- [x] Validade de triangulo
 
- Verifica se as medidas dos lados formam um triangulo
	
- [x] Classificação de Triângulos:

- Equilátero: Todos os lados iguais

- Isósceles: Dois lados iguais

- Escaleno: Todos os lados diferentes

- [x] Validação de Entrada:

- Garantia de que os valores inseridos para os lados sejam números válidos

## Como rodar o projeto

1. Clone o repostiorio

```
git clone https://github.com/gustavoduara/Classificador-de-triangulos
```

2. Navegue até a pasta raiz

```
cd Classificador-de-triangulos
```

3. Restaure as dependências

```
dotnet restore
```

4. Navegue até a pasta do projeto

```
cd ClassificacaoDeTriangulo.ConsoleApp
```
